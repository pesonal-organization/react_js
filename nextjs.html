Question 2❓

Q: types of components in nextjs?

✅ Answer

two types of components are in nextjs

  • server components
  • client component

    • Question 3❓

    Q: what is server components?

    ✅ Answer

    server components are components that render on server side and the html we write in server components , is send to browser but the JavaScript that we write in server components is not sent to browser and user only see html code output and he can not see JavaScript code out put on browser.

    all app routes components of next js are server components.

    Question 4❓

    Q: client side components in next js?

    ✅ Answer

    client side components are rendered on browser and their JavaScript code also send to browser and user can see the output/functionality of JavaScript code also on browser.

    Question 5❓

    Q: how can we make server component to client component?

    ✅ Answer

    in the top of component write 'use client'
    the server component is converted from server to client component.

    Question 6❓

    Q: when we need to make server component to client component?

    ✅ Answer

  • when we use any hook in our component like
    useEffect,useState,useRef,useContext etc
  • when we use onclick handlers like
    onclick,onChange,onSubmit
  • when we create any thing that can give response on user action
  • Add to cart button
  • Image slider
  • login/sign up form
  • cart count page that updates

    • Question 7❓

    Q: when we use server component?

    ✅ Answer

    we use server components when

    Data Fetching

    when we are fetching data by api calling

    Static/not dynamic needed

    when we have to shown only data/ui not need any event like click etc

    secret and sensitive data

    apiKeys,token, secret data keep only on server component

    Prouducts listing page

    when we only need to show all Prouducts

    Proudct details page dikhana ho

    Navbar only links
