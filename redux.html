core concept of redux


A store that holds the state of our application.
An action tells what kind of change we have to make in the state. Gives instructions increase,decrease etc.
A reducer is a function that looks at the action and updates the state according to what the action is asking.

Actions


Actions are plain JavaScript objects
Action has a type field
Action only tell what to do, but they dont tell how to do
Increment and decrement are actions

Action creator


Action creator is simply an arrow function in which we define action's object and its type

Reducer


A reducer is a function that takes the current state and an action, and gives back a new updated state.

we can create multiple reducers in our application.
Each reducer perform a specific task.
we have to register all the reducers inside a single Root Reducer.

store


The Redux store combine the state,actions and reducers.
We have only a single store in a redux application.

Root Reducer


All the reducers we collect inside Root Reducer and than we import root reducer inside our store. installation: 
npm i redux react-redux
redux and react-redux both have been installed

Provider


Provider wraps our App
It makes the redux store to available inside all components
without Provider , our components can't use redux.

useSelector


useSelector is used to get current state inside App.jsx

useDispatch


useDispatch is a React-Redux hook that lets you send (dispatch) an action to the Redux store.

App.jsx 

import { useSelector,useDispatch } from "react-redux";
import { increase,decrease } from "./actions";
const App =()=>{
  const myState = useSelector((state)=>state.ind_dec)
  const dispatch = useDispatch()
  return(
    <div>
   <h1>Welcome to react redux</h1>
   <input type="text" value={myState} /><br/>
   <button onClick={()=>dispatch(increase())}>Add</button><br/>
   <button onClick={()=>dispatch(decrease())}>Minus</button>
    </div>
  )
}
export default App;

reducer.js 


// reducer ind_dec.js 

const initialState = 0;
const ind_dec = (state = initialState,action)=>{
    switch(action.type){
        case 'INCREMENT':
            return state + 1;
            case 'DECREMENT':
                return state - 1;
                default :return state
    }

}
export default ind_dec
