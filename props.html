Question 1‚ùì

Q: what is props?

‚úÖ Answer

Props is derived from property.
It is used to transfer data from one component to another.
data sender component is called parent component and data receiver component is called child component.
we can not modify or change data of props but only we can use it.

Question 2‚ùì

Q: why do we use props?

‚úÖ Answer

we use props to
1 . To make components reuseable
2 . To pass data and functions between components
3. To manage parent-child communication.
