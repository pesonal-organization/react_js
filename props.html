Q: what is props?

‚úÖ Answer

Props is derived from property.
It is used to transfer data from one component to another.
data sender component is called parent component and data receiver component is called child component.
we can not modify or change data of props but only we can use it.

note

data can be also transferred from children component to parent components by call back. simple props // parent.jsx import React from "react"; import ChildComponent from "./child"; const ParentComponent = ()=>{ const name = 'Aziz'; const age = 24 return( <> <h1>i am parent component</h1> <ChildComponent name = {name} age = {age}/> </> ) } export default ParentComponent; ______________________________ child file // child.jsx import React from "react"; const ChildComponent = ({name,age})=>{ return( <> <h1>i am childComponent</h1> <h2>{name}</h2> <h3>My age is : {age}</h3> </> ) } export default ChildComponent; ______________________________ passing as an object data into props // parent.jsx import React from "react"; receiver file // child.jsx import React from "react"; const ChildComponent = ({person})=>{ return( <> <h1>i am childComponent</h1> <h1>My name is {person.name}</h1> <h1>My age is {person.age}</h1> <h1>My designation is {person.designation}</h1> </> ) } export default ChildComponent; Question 2‚ùì

Q: why do we use props?

‚úÖ Answer

we use props to
1 . To make components reuseable
2 . To pass data and functions between components
3. To manage parent-child communication.
