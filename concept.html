Question 20‚ùì

Q: useMemo hook and example:

‚úÖ Answer

If we have a function and we have two states, normally when any state changes the function runs again. But we don‚Äôt want the function to run on every state change because this affects performance. So we wrap the function inside useMemo. Now the function will run only when the states inside the dependency array change, and it will not run on other state changes.
useMemo remember a value ,so React does not calculate it again and again.
It is used for performance optimization

üí° Example



 import { useMemo, useState } from "react";

function UseMemo() {
  const [number, setNumber] = useState(0);
  const [count, setCount] = useState(0);

  const doubleNumber = useMemo(() => {
    console.log("Calculating...");
    return number * 2;
  }, [number]);   // Only re-run when "number" changes

  return (
    <div>
      <h1>Number: {number}</h1>
      <h2>Double: {doubleNumber}</h2>

      <button onClick={() => setNumber(number + 1)}>Increase Number</button>
      <button onClick={() => setCount(count + 1)}>Increase Count</button>
    </div>
  );
}

export default UseMemo;

Q: use of loading in finally?

‚úÖ Answer

see in exmaple:

üí° Example


       
        import React, { useEffect, useState } from "react";

const About = () => {
  const [apiData, setApiData] = useState([]);
  const [loading, setLoading] = useState(false);

  // API calling function
  const apiCallingFunction = async () => {
    const apiUrl = "https://jsonplaceholder.typicode.com/users";
    setLoading(true); // start loading

    try {
      const res = await fetch(apiUrl);
      const realData = await res.json();
      console.log(realData);

      if (realData) {
        setApiData(realData);
      }
    } catch (error) {
      console.log("Error while fetching:", error);
    } finally {
      // ye hamesha chalega: success ho ya error
      setLoading(false);
    }
  };

  // useEffect
  useEffect(() => {
    apiCallingFunction();
  }, []);

  return (
    <>
      <div>
        <div>
          {loading && <h1>data is loading...</h1>}
        </div>

        <ul>
          {apiData.map((item) => {
            return (
              <li key={item.id}>
                {item.id} : {item.name}
              </li>
            );
          })}
        </ul>
      </div>
    </>
  );
};

export default About;
