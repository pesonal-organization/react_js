Question 7❓

Q: Make a function in React-Js?

✅ Answer

see in example:

💡 Example


       // About Page
import React from "react";
import { useState } from "react";
let count = 0;
const About = ()=>{
    // we make state here
    let [count,setCount] = useState(0);
    // now we make function for increment count
    const countHandler = ()=>{
        setCount(count + 1)
    }
    // now make another function for decrement
    const decrementCount = ()=>{
        setCount(count - 1)
    }
    return(
        <div>
            <h1>About page</h1>
            <h1>count : {count}</h1>

            <button onClick={countHandler} disabled={count >=10}>Increment</button>
            <button onClick={decrementCount} disabled={count == 0}>Decrement</button>
        </div>
    )
}
export default About;

Q: what is Reconciliation in react?

✅ Answer

Reconciliation is react's smart algorithm that compares the old and new virtual DOM finds the minimal differences and applies only those change to the real DOM, it is called Reconciliation in react-js.
means our actual DOM complete new page does not load only loads the virtual dom small area that is changed by functionality that we have given.

Question 12❓

Q: what is Link in react?

✅ Answer

Link is a component from the library react-router-dom.
It is used for navigation between pages in a React app (Single Page Application).
Unlike

<a href="..."></a>, it does not reload the page. It just changes the URL and shows the correct component.

Question 15❓

Q: correct apiCall and render data on dom by map?

✅ Answer

see in example:

💡 Example


       
        import React, { useEffect, useState } from "react";

const ApiPractice = () => {
  const [users, setUsers] = useState([]);

  // apiCall function
  const apiCall = async () => {
    const apiUrl = "https://jsonplaceholder.typicode.com/users";
    try {
      const res = await fetch(apiUrl);
      const targetData = await res.json();
      setUsers(targetData);
    } catch (error) {
      console.log("something went wrong: ", error);
    }
  };

  useEffect(() => {
    apiCall();
  }, []);

  return (
    <div>
      <h1>User List</h1>
      <ul>
        {users.map((user) => (
          <li key={user.id}>{user.name}</li>
        ))}
      </ul>
    </div>
  );
};

export default ApiPractice;
