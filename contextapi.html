Q: what is context API?

âœ… Answer

Context API is a way to send data to many components where is needed without props dirlling.
The react context API has three main parts:
1.

createContext


createContext is used to create a shared place (like a board) where data can be kept so any component can read it without props.
2.

Provider


Provides gives data to all component that are wrapped inside it.
3.

useContext


by using useContext( ) any component can use data of usecontex api.

During working with contexApi
first we create createContext, we import createContex from react and store it in variable and it initial value keep null.
second we create data that we have to share among all components using contextApi.
Third we import data and createContext file in our app.jsx and we wrap our all app.jsx components inside createcontex.Provider and in value attribute we give the data file name that have we exported from datafile.
four we use data in any component where is requirement
The component in which we are using data we use useContex hook for accessing data
we import createContex from react and import createcontexfile, and inside component before return we create a variable for storing usecontext and inside usecontext parameter we keep createdcontext file and use date in component.

Q: Api call and use in multiple components using contextApi?

âœ… Answer

Now we are calling api inside App.jsx

ğŸ’¡ Example


       
        App.jsx
         
        import { BrowserRouter } from "react-router-dom";
import Navbar from "./components/navbar/navbar";
import MyRoutes from "./components/utils/routes/routes";
import createcontextfile from "./components/createcontexfile";
import collectedData from "./components/datafile";
import { useEffect, useState } from "react";

function App() {
  const [users, setUsers] = useState([]);

  // API call (ONE TIME)
  useEffect(() => {

    const fetchUsers = async () => {
      try {
        const response = await fetch("https://jsonplaceholder.typicode.com/users");
        const data = await response.json();
        setUsers(data);
      } catch (error) {
        console.log("Error while fetching users:", error);
      }
    };

    fetchUsers();

  }, []);

  return (
    <createcontextfile.Provider value={{ collectedData, users }}>
      <BrowserRouter>
        <Navbar />
        <MyRoutes />
      </BrowserRouter>
    </createcontextfile.Provider>
  );
}

export default App;

import { useEffect,useState } from "react";
import createcontextfile from "../createcontexfile";
const DataProvider = ({children})=>{
    const [users,setUsers] = useState([]);
    const [loading,isLoading] = useState(true)
    // calling api using useEffect 
    useEffect(()=>{
        const fetchUsers = async ()=>{
            try{const response = await fetch("https://jsonplaceholder.typicode.com/users");
                const jsonData = await response.json();
                setUsers(jsonData)
            }
            catch(error){ console.log(error)

            }
            finally{isLoading(false)}
        }
        fetchUsers();
    },[])
    // loader showing 
  
    return(
<createcontextfile.Provider value={{users,loading}}>
    {children}
</createcontextfile.Provider>
    )
}
export default DataProvider;
