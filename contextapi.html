Q: what is context API?

âœ… Answer

Context API is a way to send data to many components where is needed without props dirlling.
The react context API has three main parts:
1.

createContext


createContext is used to create a shared place (like a board) where data can be kept so any component can read it without props.
2.

Provider


Provides gives data to all component that are wrapped inside it.
3.

useContext


by using useContext( ) any component can use data of usecontex api.

  homeHeading: "Welcome to Home Page",
  aboutHeading: "About Us - Learn More",
  contactHeading: "Contact Us Anytime",
};

export default siteData;

second we make createContext( ) 
like 

// createContex.js file
import { createContext } from "react";

export default DataContext; 
_________________________________________________

Than we create provider 


import { BrowserRouter } from "react-router-dom"
import MyRoutes from "./components/routes/routes"
import Navbar from "./components/routes/navbar/navbar"
import DataContext from "./datafolder/createContextFolder/createContext"
import siteData from "./datafolder/datafile"
function App() {
 

  return (
   <DataContext.Provider value={siteData}>
    
    <BrowserRouter>
    <Navbar/>
    <MyRoutes/>
    
    </BrowserRouter>
   </DataContext.Provider>)
   
}

export default App

than we can use data any component we want 

home.jsx 

import React from 'react'
import { useContext } from 'react';
import DataContext from '../datafolder/createContextFolder/createContext';

const Home = () => {
    const data = useContext(DataContext)
  return (
    <div>
        <h1>{data.homeHeading}</h1>
        <p>This is homepage</p>
    </div>
  )
}

export default Home
