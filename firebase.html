Question 7❓

Q: what is collection?

✅ Answer

"collection Firebase Firestore ka ek function hai jo kisi specific collection (jaise 'Users', 'Products') ka reference return karta hai. Ye reference phir addDoc, getDocs jaise functions ke saath use hota hai us collection ke andar data add/read karne ke liye — isse data logically alag-alag groups mein organized rehta hai."

💡 Example



  // Sirf reference banaya, koi data operation nahi hua
const usersRef = collection(db, 'Users');

// Ab is reference ko use kar ke actual kaam hota hai
addDoc(usersRef, obj);      // add karna
getDocs(usersRef);          // read karna

Question 8❓

Q: what is addDoc?

✅ Answer

addDoc ek function hai jo kisi collection mein naya document add karta hai, aur Firestore khud us document ke liye unique ID generate kar deta hai.

💡 Example



  const obj = { uid: "abc123", email: "ali@gmail.com" }

const result = await addDoc(collection(db, 'Users'), obj);
console.log(result.id);  // Firestore ki generated document ID, jaise "kX9pQz7Yab"

'use client'
import React, { useState } from "react";
import {auth,db} from "@/src/firebase/firebase";
import { createUserWithEmailAndPassword } from "firebase/auth";
import { addDoc,collection } from "firebase/firestore";
const SignUpPage = ()=>{
    const [email,setEmail] = useState('');
    const [password,setPassword] = useState('');
    const signupHandler = async()=>{
        try{
            const userCredential = await createUserWithEmailAndPassword(auth,email,password)
            console.log('user has saved ', userCredential )
            
            //storing user in firebase storage
            const obj = {
                uid : userCredential?.user?.uid,
                email : email,
                password : btoa(password)
            }
            //saving data in DB 
            if(userCredential){
                const saveUser = await addDoc(collection(db,'Users'),obj)
                console.log('users save ',saveUser)
            }
            setEmail('');
            setPassword('')

        }
        catch(error){
            console.log(error)
        }
       
    }
    return(
        <div>
            <h1>Sign up Now</h1>
            <input type="email" placeholder="enter your email"
            onChange={(e)=>setEmail(e.target.value)} autoComplete="new-email"
            value={email} /><br/>
            <input type="password" placeholder="enter your password" 
            onChange={(e)=>setPassword(e.target.value)} autoComplete="new-password"
            value={password}/><br/>
            <button onClick={signupHandler}>sign up</button>
        </div>
    )
}
export default SignUpPage;

Question 10❓

Q: what is signInWithEmailAndPassword?

✅ Answer

signInWithEmailAndPassword Firebase Auth ka function hai jo existing user ko login karwata hai email aur password ke through — matlab ye check karta hai ki user pehle se registered hai ya nahi, aur agar credentials sahi hain to login kar deta hai.

💡 Example


        
        'use client'
import React, { useState } from "react";
import {auth} from "@/src/firebase/firebase";
import { signInWithEmailAndPassword } from "firebase/auth";
const LogIn = ()=>{
    const [email,setEmail] = useState('');
    const [password,setPassword] = useState('');
    const logInHandler = async()=>{
        try{
            const userCredential = await signInWithEmailAndPassword(auth,email,password)
           alert('login successfully')
            setEmail('');
            setPassword('')
        }
        catch(error){
            console.log(error.message)
        }
       
    }
    return(
        <div>
            <h1>Log In</h1>
            <input type="email" placeholder="enter your email"
            onChange={(e)=>setEmail(e.target.value)} autoComplete="new-email"
            value={email} /><br/>
            <input type="password" placeholder="enter your password" 
            onChange={(e)=>setPassword(e.target.value)} autoComplete="new-password"
            value={password}/><br/>
            <button onClick={logInHandler}>login</button>
        </div>
    )
}
export default LogIn;

Q: Difference b/w createUserWithEmailAndPassword and signInWithEmailAndPassword?

✅ Answer

createUserWithEmailAndPassword
creates a new user and is used in sign up component
signInWithEmailAndPassword
is used for logging in existed user that is already signed up.
