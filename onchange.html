Q: what is onChange Event?

âœ… Answer

onChange is an event and it is used in react.
when we write anything in input, onchange sees what we typed.
onChange calls useState to store it that have we defined.
now our typing is storing in the variable
react can show it anywhere in the page.

ğŸ’¡ Example


       
        import { useEffect, useRef, useState } from "react";
const UseStateHook = ()=>{
    const [item,setItem] = useState('');
    const [name,setName] = useState('');
    const inputOne = useRef();
    const inputTwo = useRef();
    // function for clearing input
    function removing(){
        inputOne.current.value = '';
        inputTwo.current.value = '';
    }
    // useeffect for applying focus on inputOne on page loading
    useEffect(()=>{
        inputOne.current.focus();
    },[])

    return(
       <>
        <h1>use ref hook</h1>
       <input type="text" value={item} onChange={(event)=>setItem(event.target.value)} ref={inputOne} /><br/>
       <input type="text" value={name} onChange={(event)=>setName(event.target.value)} ref={inputTwo} />
       <button onClick={removing}>clear inputs</button>
       <h2>item : {item}</h2>
       <h2>name : {name}</h2>
        
       </>
        
    )
}
export default UseStateHook;
