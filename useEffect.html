Question 6â“

Q: what is useEffect( )?

âœ… Answer

ğŸ‘‰ useEffect is a React Hook that lets you run code when your component renders or when something changes.
useEffect is used for side effects like fetching data, updating the DOM, running code when something changes, or cleaning up resources.

ğŸ’¡ Example


       
        ğŸ”¹ Uses of useEffect

Run code when the component first shows up (like page load).

useEffect(() => {
  console.log("Component mounted");
}, []);


Fetch data from an API

useEffect(() => {
  fetch("https://api.example.com/data")
    .then(res => res.json())
    .then(data => console.log(data));
}, []);


Update something outside React (like page title or console)

useEffect(() => {
  document.title = "New Title";
}, []);


Run code when a value changes

useEffect(() => {
  console.log("Count changed");
}, [count]); // runs when count changes


Set up and clean up (timers, event listeners, etc.)

useEffect(() => {
  const timer = setInterval(() => console.log("Tick"), 1000);

  return () => clearInterval(timer); // cleanup
}, []);

function Watch() {
  const [time, setTime] = useState(0);

  useEffect(() => {
    // start interval
    const timer = setInterval(() => {
      setTime((prev) => prev + 1);
    }, 1000);

    // cleanup (cancellation process)
    return () => {
      clearInterval(timer);
    };
  }, []); // empty array â†’ runs only once

  return <h1>{time} seconds</h1>;
}
