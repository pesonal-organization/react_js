Question 1❓

Q: what is life Cycle Method in react?

✅ Answer

life cycle method in react is same like human life ,birth, grow up and death
🧍‍♂️ Human Life ↔ ⚛️ React Component Life

1️⃣ Birth → Mounting

Human:
A child is born into the world

React:
A component is created
A component is rendered on the screen

👉 This phase is called Mounting

Component appears in the DOM

Data can be fetched

Setup happens
API calls can be made
Event listeners can be added

Class component example:
componentDidMount()

Functional component example:
useEffect(() => { }, [])

2️⃣ Growth → Updating

Human:
The child grows
The child changes with time

React:
Component updates when state changes
Component updates when props change

👉 This phase is called Updating

Component re-renders

UI changes according to new data

Class component example:
componentDidUpdate()

Functional component example:
useEffect(() => { }, [state, props])

3️⃣ Death → Unmounting

Human:
Life comes to an end

React:
Component is removed from the screen

👉 This phase is called Unmounting

Component is removed from the DOM

Cleanup is performed

Timers are cleared

Event listeners are removed

API calls are cancelled

Class component example:
componentWillUnmount()

Functional component example:
useEffect(() => {
return () => { }
}, [])

💡 Example


       
        App.jsx or the component in which we call life cycle component
        ________________________________________________________________

        import { useState } from "react"
import LifeCycleMethod from "./lifecycle"

function App() {
  const [display,setDisplay] = useState(true)

  return (
    <>
      <h1>app</h1>
      {
        display?<LifeCycleMethod/>:null
      }
      <br/>
      <button onClick={()=>setDisplay(!display)}>Hide/Display</button>
    </>
  )
}

export default App

LifeCycleMethod component 
__________


import React, { useEffect, useState } from "react";
const LifeCycleMethod = ()=>{
    const [counter,setConter] = useState(1)
    // function whenMount 
    const whenMount = ()=>{
        console.log('component mounted successfully')
    }
    // fuction whenUpdate 
    const whenUpdate = ()=>{
        console.log('component has updated')
    }
    // function whenEnd 
    const whenEnd = ()=>{
        console.log('component unmounted')
    }
    // useeffect for function whenMount 
    useEffect(()=>{
        whenMount();
    },[])
// useeffect for whenUpdate 
useEffect(()=>{
    whenUpdate()
},[counter])
// useeffect for whenEnd function
useEffect(()=>{

    return()=>whenEnd()
},[])
    return(
        <>
        <h1>Life Cycle Method</h1>
        <h1>value : {counter}</h1>
        <button onClick={()=>setConter(counter + 1)}>increase value</button>
        </>
    )
}
export default LifeCycleMethod;
