Q: what is useEffect hook?

âœ… Answer

useEffect hook run when a component renders.
useEffect is used to handle sideEffects
like
API Call krna
Data Fetch krna
SetInterval setTimeout use krna
if we want that some code render when our state changes like counter than we write same code inside useEffect,
The code that we write inside useEffect runs one time must when page loads.

ğŸ’¡ Example


       
        import React, { useEffect, useState } from 'react'

const Contact = () => {
 const [counter,setCounter] = useState(0)
 const [username,setUserName] = useState('')
 
//  updating counter function 
const updatingCounter = ()=>{
  console.log('counter is updating')
}
  // updating name function
const updatingName = ()=>{
  console.log('name is updating')
}
   
// useEffect for updatingCounter 
useEffect(()=>{
  updatingCounter();
},[counter])
  
  // useEffect for updatingName 
  useEffect(()=>{
    updatingName()
  },[username])

  return (
  <div>
    <h1>use effect</h1>
    <h1>{(counter)?(counter) : ('No counter')}</h1>
    <h2>username : {username?username : 'No username existing'}</h2>
    
    <button onClick={()=>setCounter(counter + 1)}>Increment</button>
    <button onClick={()=>setUserName('Abid')}>update UserName</button>
  </div>
  )
}

export default Contact

Question 4â“

Q:what is useReducer Hook?

âœ… Answer

useState for simple state management
useReducer for complex state management
useReducer hook is same like useState Hook
in small website we should use useState hook but it is highly recommended that in complex website we must use useReducer hook.
useReducer works with reducer and reducers are pure function
useReducer is mostly used with react-redux libaray.
It is a hook that is used for stateManagement
It is an alternate of useState
useState is built using useReducer
Every task that we can perfom by useState we can also do it by useReducer

ğŸ’¡ Example


       
        // reducer.jsx
import React, { act } from "react";
import { useReducer } from "react";
 const Reducer = ()=>{
    const initialState = 0;
    const reducer = (state,action)=>{
    
        switch(action){
            case 'increment':
                return state + 1
                case 'decrement':
                    return state - 1
                    case 'reset' :
                        return initialState 
                        default : 
                        return state
        }

    }
    const [count,dispatch] = useReducer(reducer,initialState);
    return(
        <div><h1>Reducer</h1>
        <h1>count = {count}</h1>
        <button onClick={()=>dispatch('increment')}>Increment</button>
        <button onClick={()=>dispatch('decrement')}>Decrement</button>
        <button onClick={()=>dispatch('reset')}>Reset</button>
        
        </div>
    )
 }
 export default Reducer;

âœ… Answer

The hooks that we creates ourselves are called custom hooks.
Rules for creating custom hooks
1. prefix with use
custom hook is starts with the word of use
2. use Built in hooks

    const [count,setCount] = useState(0);
